## Nuevo: plantilla de sistema multi-agente auditable
Se agregó la carpeta `openclaw-enterprise/` con una base para construir un sistema empresarial multi-agente con:
- separación de memoria financiera/técnica/auditoría,
- workflow de 7 estados con HITL,
- políticas por agente,
- prompts base,
- SQL para auditoría y trazabilidad.

Empieza por `openclaw-enterprise/README.md` y luego ejecuta `openclaw-enterprise/scripts/bootstrap.sh`.
